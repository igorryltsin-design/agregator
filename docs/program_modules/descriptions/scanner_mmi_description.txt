Сканер многомодальной информации автоматизирует обход каталогов, извлекает признаки объектов, рассчитывает сигнатуры и формирует граф близости между найденными элементами. Программа применяет набор проб, поддерживает очередь заданий, собирает статистику по времени обработки и сохраняет результаты для последующего анализа. Такой подход обеспечивает равномерную сигнализацию по всем типам файлов при отсутствии жестко заданных категорий.

Система дополняет первичное сканирование последующей регуляцией: регулирует веса многомодальных сигналов, профилирует источники и архивирует сводные отчеты. Полученные структуры и отчеты готовятся к передаче в регистрационные материалы как самостоятельный продукт для мониторинга и инвентаризации многомодальной среды.
