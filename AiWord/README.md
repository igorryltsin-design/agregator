# AiWord - редактор научных текстов

`AiWord` - фронтенд-приложение для подготовки научных материалов с поддержкой AI-помощника, шаблонов и работы с библиографией.  
В составе Agregator доступно как интегрированный модуль, а также может запускаться отдельно.

## 1. Назначение

`AiWord` помогает:

- формировать структуру научного текста;
- генерировать и редактировать черновики;
- поддерживать единый стиль и критерии;
- работать с цитированием и списком литературы;
- экспортировать результат в удобные форматы.

## 2. Основные возможности

- агентные сценарии: план, черновик, улучшение, рецензия;
- профили критериев (академические/прикладные);
- markdown-редактор с предпросмотром;
- библиография по `.bib` и ссылкам вида `[@citekey]`;
- экспорт в `.docx` и `.md`;
- автосохранение проекта и восстановление состояния;
- темная/светлая темы и горячие клавиши.

## 3. Требования

- Node.js 18+ (рекомендуется 20+)
- npm
- доступ к LLM endpoint-у (для AI-функций)

## 4. Установка и запуск

```bash
npm install
npm run dev
```

По умолчанию интерфейс открывается на `http://localhost:5173`.

## 5. Сборка production

```bash
npm run build
npm run preview
```

## 6. Интеграция с Agregator

При интеграции в основной проект:

- `AiWord` получает конфигурацию AI из Agregator;
- может использовать данные каталога и BibTeX источники;
- доступен внутри общего интерфейса по роуту `/aiword`.

## 7. Работа с проектом

Рекомендуемый цикл:

1. Создать структуру документа.
2. Заполнить разделы черновиком.
3. Улучшить текст по выбранным критериям.
4. Проверить ссылки и список литературы.
5. Экспортировать финальную версию.

## 8. Типовые проблемы

- **Нет AI-ответа:** проверить URL модели и ключ доступа.
- **Проблема с экспортом:** проверить корректность markdown и таблиц.
- **Не подтягивается библиография:** проверить формат `.bib` и citekey.
